Conocimiento técnico previo requerido

  • Programación orientada a objetos básica
    • Qué es un Objeto?
    • Qué es una Clase?
    • En qué consiste la Encapsulación? Agregación? Composición?
    • Qué es la Herencia (inheritance)?
    • Qué es una Clase Abstracta?
    • Cuál es la diferencia entre una Interfaz y una Clase Abstracta?
    • Qué es un Class Diagram?
    • Qué estructuras de datos conoces y has manejado?
  • Patrones de diseño
    • En qué consiste el patrón Model-View-Controller (MVC)?
    • Cuál es la diferencia entre un Modelo y un Controlador?
  • APIs
    • Qué es un API?
    • Cuáles son las ventajas de tener un API?
  • HTML/CSS
    • Qué es un tag de HTML? Qué es un elemento?
    • Qué es un atributo?
    • Qué es el DOM?
    • Qué es un selector de CSS?
    • Cuál es la diferencia entre un elemento HTML inline y uno block?
    • Cómo se modifican estilos básicos de un elemento HTML con CSS (color de fondo, color de texto, etc.)?
  • JavaScript
    • Para qué se utiliza JavaScript en una página web?
    • Cómo se pueden asociar funciones a eventos como un clic en un elemento HTML?
    • Cómo se pueden crear Clases y Objetos en JavaScript?
  • Ruby
    • ¿Qué es un símbolo?
    • ¿Como se lleva a cabo la sobrescritura de métodos en clases?
  • Recursos en Internet
    • Cómo funciona StackOverflow?
    • Qué diferencias conoces entre SaaS, PaaS, IaaS?

Práctica de tareas básicas de manejo de código

  • Git
    • Crear un repo local (git init)
    • Hacer un commit (git commit)
    • Hacer un branch (git branch)
    • Hacer merge entre dos branches (git merge)
    • Hacer reset o revert hacia un commit anterior (git reset)
  • Github
    • Crear un repo en github
    • Sincronizar en computadora local
    • Hacer un commit y push
  • Command Line Interface
    • Crear un directorio local (mkdir)
    • Cambiar de directorio (cd) y listar sus contenidos (ls)
    • Crear un symlink (ln -s)
    • Eliminar un archivo (rm)
    • Modificar permisos de un archivo o directorio (chmod)
