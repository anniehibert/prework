Práctica de tareas básicas de manejo de código

  • Git
    • Crear un repo local (git init)
    • Hacer un commit (git commit)
    • Hacer un branch (git branch)
    • Hacer merge entre dos branches (git merge)
    • Hacer reset o revert hacia un commit anterior (git reset)
  • Github
    • Crear un repo en github
    • Sincronizar en computadora local
    • Hacer un commit y push
  • Command Line Interface
    • Crear un directorio local (mkdir)
    • Cambiar de directorio (cd) y listar sus contenidos (ls)
    • Crear un symlink (ln -s)
    • Eliminar un archivo (rm)
    • Modificar permisos de un archivo o directorio (chmod)
